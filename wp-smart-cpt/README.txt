== Description ==

WP Smart CPT Generator is a WordPress plugin that allows users to easily create custom post types (CPTs) and manage their information through an intuitive admin dashboard. This plugin provides a user-friendly interface to create and manage CPTs without writing any code.

== Installation ==

Download the WP Smart CPT Generator plugin from the WordPress Plugin Directory.
Upload the plugin files to the /wp-content/plugins/wp-smart-cpt-generator directory, or install the plugin directly from the WordPress Plugins page.
Activate the WP Smart CPT Generator plugin through the 'Plugins' menu in WordPress.
== Frequently Asked Questions ==

= How do I create a custom post type (CPT)? =

To create a CPT using WP Smart CPT Generator:

Go to the CPT Generator menu on the WordPress admin dashboard.
Fill out the form with the required CPT information (name, singular name, description, and menu).
Submit the form to create the CPT.
= How can I delete a CPT? =

To delete a CPT:

Go to the CPT Generator menu on the WordPress admin dashboard.
Switch to the detailed information tab.
Locate the CPT you want to delete and click the delete button.
